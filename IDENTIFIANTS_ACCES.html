Inviter l'administrateur sur Sanity

Étapes :
  1. Se connecter sur https://www.sanity.io/manage
  2. Ouvrir le projet AMSP
  3. Cliquer sur "Members" dans le menu gauche
  4. Cliquer sur "Invite member"
  5. Entrer l'adresse email de l'administrateur
  6. Choisir le rôle "Administrator"
  7. Cliquer sur "Send invite"
⚠️ Le plan gratuit Sanity ne propose que deux rôles : Administrator (peut tout faire) et Viewer (lecture seule). Donner le rôle Administrator est nécessaire pour que l'admin puisse modifier le contenu.

L'administrateur pourra modifier textes, photos et actualités, mais aussi inviter d'autres personnes ou modifier les paramètres du projet. Faire une sauvegarde des données avant de lui donner accès (voir section Sauvegarde ci-dessous).
💡 L'administrateur devra créer son propre compte Sanity (gratuit) à l'adresse https://www.sanity.io avec l'email auquel l'invitation a été envoyée.

Sauvegarde du contenu Sanity

Avant de donner les accès et en cas d'erreur, exporter le contenu avec cette commande dans le terminal (depuis le dossier du projet) :

npx sanity dataset export production backup-amsp.tar.gz

Le fichier backup-amsp.tar.gz contient tout le contenu du site (textes, références images, actualités…). Pour restaurer :

npx sanity dataset import backup-amsp.tar.gz production --replace
⚠️ Ces commandes sont à exécuter depuis le dossier amsp-studio (le projet Sanity), pas depuis le dossier du site React. Conserver le fichier de sauvegarde sur votre ordinateur ou dans un cloud (Google Drive, etc.).
